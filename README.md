# .github
Stake резервный домен предоставляет доступ к платформе через альтернативный адрес, если основной сайт заблокирован или недоступен. Через рабочую ссылку можно войти в личный кабинет, пополнить баланс, играть в казино, получать бонусы и обращаться в поддержку. Работает на всех устройствах без VPN и дополнительных настроек.
